# Computer Vision Module (HSLU CAS ML)

This repository contains the material for the Computer Vision module for the CAS ML @ HSLU.

Lots of material can be found in the online book (by Umberto Michelucci) [https://adl.toelt.ai](https://adl.toelt.ai). 

Especially interesting are the following notebooks that can be opened directly in Google Colab for testing.

- Convolution Operator [https://colab.research.google.com/github/toelt-llc/ADL-Book-2nd-Ed/blob/master/docs/CNN/The_convolution_operator.ipynb](https://colab.research.google.com/github/toelt-llc/ADL-Book-2nd-Ed/blob/master/docs/CNN/The_convolution_operator.ipynb)
- Multi-class Classification with CNNs [https://colab.research.google.com/github/toelt-llc/ADL-Book-2nd-Ed/blob/master/docs/CNN/Multiclass_classification_with_convolutional_neural_networks.ipynb](https://colab.research.google.com/github/toelt-llc/ADL-Book-2nd-Ed/blob/master/docs/CNN/Multiclass_classification_with_convolutional_neural_networks.ipynb)
